**Os índices das questões são clicáveis e levam à resolução do exercício.**

[1.](1.py) Implemente um sistema que manipule as notas dos alunos de uma turma. No início da execução do programa deve ser perguntado ao usuário que cadastra as notas, o número de alunos da turma.
Restrição: 
    0 <= Nota <= 10
Ações esperadas:
- Indicar a média de notas da turma;
- Indicar a maior nota;
- Indicar a menor nota.
- Retorna a diferença entre a maior e a menor nota.
Considerar que não existem notas iguais.


[2.](2.py) Faça um programa que leia uma série de números inteiros e os armazene em uma lista. Em seguida, o programa deve determinar se nessa série de valores aparece algum valor repetido. Mostrar mensagem dizendo se existe ou não um número repetido!


[3.](3.py) Elabore um programa que leia um array de números inteiros com “N” elementos reais e mostre as posições que o maior e o menor elemento ocupam a lista.


[4.](4.py) Escreva um programa que leia uma lista X de N elementos numéricos e um valor numérico qualquer K. Após, determine e imprima a lista resultante da multiplicação de K por X.


[5.](5.py) Faça um programa que leia uma lista X[0..19] e, após, troque a posição do 1º elemento pela posição do 20º elemento, o 2º com o 19º e assim por diante. Ao final, imprima X.
